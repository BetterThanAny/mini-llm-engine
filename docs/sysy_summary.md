# SysY 编译器项目总结

> 面试前 5 分钟讲解稿。`[填]` 处按个人实现补全。

## 一句话简介

用 C++ 实现的 SysY 语言编译器,前端完成词法/语法/语义分析,中端生成 Koopa IR,后端输出 RISC-V 汇编,支持函数调用、数组、控制流、常量传播等优化。

## 编译流水线

```
源码 (SysY)
  └─ Lexer (flex / 手写)        → Token 流
  └─ Parser (bison / 手写)      → AST
  └─ 语义分析 + 符号表           → 带类型标注的 AST
  └─ IR 生成                    → Koopa IR (SSA)
  └─ 优化 pass                  → 优化后的 Koopa IR
  └─ 指令选择 + 寄存器分配       → RISC-V 汇编
```

### 后端
- **指令选择**:[填: 一对一翻译 / 简单匹配模式]
- **寄存器分配算法**:[填: 线性扫描 / 图着色 / 简单贪心 + 溢出到栈]
- **函数调用约定**:遵循 RISC-V calling convention,a0-a7 传参,caller/callee saved 寄存器分别处理
- **栈帧布局**:
  ```
  高地址  ┌──────────────┐
          │ 保存的 ra/s0 │
          │ callee saved │
          │ 局部变量/alloca │
          │ outgoing args │
  低地址  └──────────────┘ ← sp
  ```
